Live Location Sharing App

About the App

This app allows users to share their live location within a group in real-time. Built using Python (Django) for the backend and Dart (Flutter) for the mobile app, it integrates Google Maps APIs for accurate location tracking. The app is fully compliant with Google Play Store policies and is hosted on Namecheap.

Technology Stack

  • Backend: Python (Django)
  • Frontend: Dart (Flutter)
  • APIs: Google Maps (Licensed)
