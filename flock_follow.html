About the App

The app lets users share their live location with friends or within groups in real-time. It features a robust Python (Django) backend and a sleek Flutter (Dart) mobile interface. Using Google Maps APIs ensures accurate tracking. It’s Play Store–policy compliant and can be hosted on Namecheap.

Technology Stack

  • Backend: Python (Django)
  • Frontend: Flutter (Dart)
  • APIs: Google Maps (Licensed)
